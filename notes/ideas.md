hue → pitch, brightness → velocity, saturation → note duration

A reasonable mapping to start with:

  ┌────────────────────────┬─────────────────────────────┐
  │     Color Property     │      Musical Property       │
  ├────────────────────────┼─────────────────────────────┤
  │ Hue (0–360°)           │ Pitch / scale degree        │
  ├────────────────────────┼─────────────────────────────┤
  │ Brightness/Value       │ Velocity (volume)           │
  ├────────────────────────┼─────────────────────────────┤
  │ Saturation             │ Note duration or density    │
  ├────────────────────────┼─────────────────────────────┤
  │ Dominant color palette │ Key / mode (major vs minor) │
  └────────────────────────┴─────────────────────────────┘

For example: warm hues (reds/oranges) → higher notes, cool hues (blues) → lower notes. Low saturation → sparse, slow melody. High saturation → dense, faster rhythm.
